智能体学习档案 - 分享版
========================

使用方法
--------
1. 请先完整解压压缩包，不要直接在压缩包内运行。
2. Windows 双击“启动学习网站.bat”。
3. 浏览器会自动打开 http://localhost:5000。
4. 关闭命令窗口即可停止网站。

首次运行
--------
- 电脑需要安装 Python 3.9 或更高版本。
- 启动器会在 web_demo/.venv 创建独立环境，不会修改全局 Python 包。
- 首次运行会联网安装 Flask，之后通常不需要联网。
- 安装完成后，再次运行通常不需要联网。

网站内容
--------
- 16 章智能体学习笔记
- 五阶段学习路线
- 笔记搜索与阶段筛选
- Markdown 长文阅读器
- 阅读进度与完成状态
- 本地超级智能体学习助手

目录说明
--------
- notes：网站显示的 Markdown 学习笔记
- code：网站使用的纯本地学习助手
- web_demo：网站前后端

隐私说明
--------
- 分享包不含 API 密钥、账号密码或原作者的浏览器学习进度。
- 网站仅监听本机 127.0.0.1，不开放局域网访问。
- 网站只使用本地能力，不会调用或消耗任何 AI API 额度。
- 阅读完成状态只保存在使用者自己的浏览器中。
- 学习助手的会话记忆仅保留到服务关闭，不会写入分享目录。

内容更新
--------
直接修改 notes 目录中的 Markdown 文件，刷新网页即可看到更新。
